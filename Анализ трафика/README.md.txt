## Задание 1. Исследование сетевых пакетов (SMB-анализ)

### Задача
Проанализировать дамп трафика и ответить на вопросы о SMB-соединении.

### Результаты

#### 1. Опиши общими словами, что происходит в этом дампе.

Устанавливается SMB2-соединение между клиентом и сервером. Клиент запрашивает доступ к файлам, используется шифрование.

**Как нашёл:** Фильтр `smb2` → пакеты Negotiate → Session Setup → Tree Connect → Read.

---

#### 2. Какая версия SMB-диалекта использовалась в соединении.
**Ответ:** 3.1.1

**Как нашёл:** В пакете `Negotiate Protocol Request` → поле `Dialect Revision` → 0x0311.

---

#### 3. Предоставь session ID установленного соединения.
**Ответ:** `0x0000281cac000069`

**Как нашёл:** Фильтр `smb2.cmd == 1` → поле `Session ID` в ответе.

---

#### 4. Какая учетная запись участвовала в соединении.
**Ответ:** `administrator`

**Как нашёл:** В пакете `Session Setup Request` → поле `Account Name`.

---

#### 5. Выпиши MAC-адреса действующих сторон.

 Клиент  `55:54:00:d1:44:45` 
 Сервер  `52:54:00:07:03:2d` 

**Как нашёл:** Ethernet-заголовок (`eth.src`, `eth.dst`).

## Задание 3. Использование Scapy (Python)

### Задача
Написать скрипт на Python с использованием Scapy, который отправляет TCP-пакет на локальный порт 127.0.0.1:12345 с сообщением:

> *"Dear Steel Cat! This is no attack, it's my hamster Pinkie you should track"*

Затем перехватить этот пакет в Wireshark.

### Результат

**Код скрипта (`script_scapy.py`):**
```python
from scapy.all import *

def send_message():
    target_ip = "127.0.0.1"
    target_port = 12345
    message = "Dear Steel Cat! This is no attack, it's my hamster Pinkie you should track"

    packet = IP(dst=target_ip) / TCP(dport=target_port) / message
    send(packet, verbose=False)

if __name__ == "__main__":
    send_message()
