# Защита операционной системы Linux

## Цель проекта
Настроить безопасность ОС Linux с использованием встроенных средств: bash-скриптов, iptables, SSH и системы мониторинга.

---

## Что было сделано

### 1. Сбор информации о системе
- Разработан bash-скрипт `get_all_info.sh`, который собирает:
  - Установленные пакеты
  - Запущенные процессы
  - Открытые порты
  - Версию ядра и ОС
- Скрипт автоматически устанавливает пакеты `cowsay` и `sl`
- Результат сохраняется в файл `info` и упаковывается в архив `OS_RESULT.tar`

**Результат:** [`results/info`](./results/info)

---

### 2. Настройка мониторинга
- Настроен `logwatch` для ежедневного сбора логов
- Настроен почтовый сервер `postfix` для отправки отчётов через Gmail (SMTP)
- Добавлена задача в `crontab` (ежедневно в 00:05) — отправка логов на почту

---

### 3. Защита системы (iptables + SSH)
- Настроен SSH-доступ с использованием **RSA-ключей** (вход по ключу, вход по паролю отключён)
- Запрещены подключения по порту `80/tcp` извне (iptables)
- Настроен файл `sshd_config` для безопасного подключения

---

## Использованные технологии
`Linux (Kali)`, `Bash`, `iptables`, `SSH (RSA-ключи, Ed25519)`, `postfix`, `logwatch`, `cron`,

---

- Безопасно настраивать SSH-доступ (ключи вместо паролей, алгоритм Ed25519)
- Создавать правила iptables для ограничения доступа к портам
