# Основы Windows — Настройка безопасности операционной системы

## Цель проекта
Настроить безопасность ОС Windows с помощью встроенных средств: групповых политик, BitLocker, брандмауэра и PowerShell-скриптов.

---

## Что было сделано

### 1. Настройка политик безопасности
- Установлена минимальная длина пароля — **8 символов**
- Включено требование к сложности пароля (спецсимволы, разные регистры, цифры)
- Установлен срок действия пароля — **30 дней**
- Разработан PowerShell-скрипт [`get_sec_pol`](./скрипты/get_sec_pol), который собирает информацию о политиках безопасности

**Результат:** [`secpol.txt`](./результаты%20скриптов/secpol.txt)

---

### 2. Шифрование томов (BitLocker)
- Зашифрован системный том с использованием **BitLocker**
- В качестве доступа к ключу использован пароль
- Ключ расшифровки сохранён (на флешку и в бумажном виде)

**Результат:** [`BitLocker.txt`]

---

### 3. Настройка прав пользователей
- Запрещён вход в систему по сети для пользователя **Guest**
- Разрешено делать бэкапы только группе **«Администраторы»**
- Запрещено выключение компьютера без входа в систему
- Запрещено подключение по RDP для группы **«Пользователи»**

---

### 4. Настройка брандмауэра Windows
- Заблокированы входящие соединения на порт **80** (HTTP) — правило `Block_http_conn`
- Разрешены входящие подключения на порт **3389** (RDP) — правило `Allow_rdp_conn`
- Запрещены исходящие подключения на порт **21** (FTP) — правило `Block_ftp_conn`
- Запрещены входящие ICMP-пакеты (ping) — правило `Block_ping_conn`
- Разработан PowerShell-скрипт [`get_fw_rules`](./скрипты/get_fw_rules), который выводит только созданные правила

**Результат:** [`result.txt`](./результаты%20скриптов/result.txt)

---

## Использованные технологии
`Windows 10 Pro`, `PowerShell`, `BitLocker`, `Group Policy Editor (gpedit.msc)`, `Windows Firewall`, `UAC`
